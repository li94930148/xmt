# XMT v2.10.0

- 版本号：`v2.10.0`
- 更新时间：2026-07-23
- 升级类型：功能升级
- 功能模块：Creator Data Center、账号驾驶舱、作品分析、粉丝分析、运营报告

## 数据库变化

- 新增 `creator_work_analysis`、`creator_trend_snapshots`、`creator_reports` 及对应索引。
- 新增 `creator:report:view`、`creator:report:manage` 权限及当时文档中声明的角色绑定。

## API变化

- 保持 `POST /api/creator-agent/data-sync` 兼容；新增 Creator Data Center 查询、分析与报告能力所需的服务端接口。

## 前端变化

- 增加数据驾驶舱、作品库、作品复盘、作品详情、趋势、粉丝和运营报告页面入口。

## 修复问题

- 将 Creator Data Center 明确为只使用已入仓平台数据的独立分析中心，避免混用选题、创作、发布或员工日报数据。

## 风险说明

- 分析结果依赖真实采集字段的完整度；缺失指标不会由外部 AI 或模拟数据补齐。
- 本说明依据提交 `3bb0751`、`CHANGELOG.md`、`docs/Creator-Data-Center-v2.10.0.md` 和 `docs/migrations/v2.10.0.sql` 补齐。
