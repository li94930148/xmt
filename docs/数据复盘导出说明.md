# 数据复盘导出说明

## 支持格式

- Markdown：`GET /api/retrospectives/:id/export?type=markdown`
- HTML 打印页：`GET /api/retrospectives/:id/export?type=html`

前端在详情页提供“导出 Markdown”和“打印页”两个入口。

## 导出内容

导出包含：

- 标题
- 类型和模板
- 周期
- 状态
- 负责人
- 创建人
- 发布时间
- 归档时间
- 复盘结论
- 指标快照
- 行动项及处理结果

## 权限

导出接口复用复盘详情查看权限。无法查看复盘的用户不能导出。

## 安全与限制

- HTML 导出会对文本内容做转义。
- 本轮不生成 PDF。
- 不读取外部平台数据。
- 不把 token、密钥或登录态写入导出内容。
