# 数据复盘系统前端设计

## 当前页面

- `/retrospectives`：复盘中心列表页，包含筛选、新建入口、复盘列表和“我的复盘行动项”。
- `/retrospectives/:id`：复盘详情页，包含基础信息、指标快照、结论、日报风险联动、行动项、发布归档和导出入口。
- `/analytics`：继续作为实时数据看板保留，不与复盘中心合并。
