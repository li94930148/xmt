# 编辑器主题回归检查清单

## 浅色模式

- [ ] 编辑器正文可读，默认正文使用主题变量或语义色。
- [ ] 工具栏按钮、边框、hover 状态可读。
- [ ] 字号下拉层、颜色下拉层、弹窗层级清楚。
- [ ] 表格、引用、代码块、mark 和 comment-mark 有足够对比。
- [ ] 历史版本预览不出现不可读黑字或白字。

## 暗色模式

- [ ] 编辑器正文不出现默认黑字。
- [ ] 工具栏不出现白底黑字。
- [ ] 下拉层和弹窗不出现白底黑字。
- [ ] 历史版本和富文本预览使用暗色兼容样式。
- [ ] 旧 HTML 中默认黑色正文可被兼容处理。
- [ ] 用户主动选择的红、蓝、绿、黄、紫等强调色不被误清洗。
