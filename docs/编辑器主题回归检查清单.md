# 编辑器主题回归检查清单

## 浅色模式

- [ ] 编辑器正文：正文使用 `var(--editor-fg)`，背景层级清楚。
- [ ] 工具栏：按钮、边框、hover 状态可读。
- [ ] 字号下拉：面板、选中态、hover 状态可读。
- [ ] 颜色下拉：色板边框可见，默认色不写死黑色。
- [ ] 表格：边框、表头背景、正文可读。
- [ ] 引用：左边线和次级文字有层级。
- [ ] 代码块：块级代码和行内代码背景与正文区区分。
- [ ] 历史版本预览：富文本内容不丢样式。
- [ ] MarkdownPreview：正文、代码、引用可读。
- [ ] 富文本预览：`dangerouslySetInnerHTML` 外层带 `editor-content-preview`。

## 暗色模式

- [ ] 编辑器正文不能出现默认黑字。
- [ ] 工具栏不能白底黑字。
- [ ] 下拉层不能白底黑字。
- [ ] 历史版本不能暗底黑字。
- [ ] 旧 HTML 中 `#000000/#333333/rgb(0,0,0)/rgb(51,51,51)` 可读。
- [ ] 红、蓝、绿、黄、紫等用户强调色不能被错误清洗。
- [ ] 批注、mark、高亮在暗色背景下仍可辨认。
- [ ] 表格、引用、代码块边界清楚。

## 业务页面

- [ ] 新增选题：编辑和预览都可读。
- [ ] 选题详情：大纲编辑和只读预览可读。
- [ ] 创作详情：当前版本编辑和历史版本预览可读。
- [ ] 发布详情：剧本编辑和只读预览可读。
- [ ] 资源归档：稿件内容预览可读。
- [ ] 历史版本区域：版本切换后仍使用主题兼容层。
